
# HRM Management System

A modern Human Resource Management web application built using **React.js** and **Django**.

## 🚀 Live Demo

https://hrm-admin-panel.netlify.app

## 📌 Features

* Admin Login Authentication
* Dashboard with Statistics
* Department Management

  * Add Department
  * Edit Department
  * Delete Department
* Employee Management

  * Add Employee
  * Edit Employee
  * Delete Employee
  * Search Employee
* Responsive UI Design
* Deployed Online

## 🛠 Tech Stack

### Frontend

* React.js
* JavaScript
* CSS

### Backend

* Django
* Django REST API

### Database

* SQLite

## 📂 Installation Steps

### Frontend

```bash

```

### Backend

```bash
pip install -r requirements.txt
python manage.py runserver
```
